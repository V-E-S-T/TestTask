# TestTask
Создать небольшое тестовое full stack приложение, реализовать CRUD для
модели:
 User {
  userId,
  firstName,
  lastName,
  birthDay,
  gender
 }
 
 Requirements:
 
 1) Получать данные 
 2) Возможность добавить запись
 3) Возможность удалить запись
 4) Возможность редактировать запись
 
 Back-end:
 1) REST api для всех операций
 2) Организовать работу с БД
 Использовать Hibernate, PostreSQL, Spring Framework
